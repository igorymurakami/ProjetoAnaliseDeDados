# Cargos na Área de Dados

A área de dados cresceu exponencialmente nos últimos anos, e com isso, os papéis se tornaram mais especializados. Os três principais cargos que formam o núcleo de muitas equipes de dados são:

## Analista de Dados (Data Analyst)

O Analista de Dados é focado em extrair insights valiosos de dados já existentes para apoiar a tomada de decisão do negócio.

*   **Responsabilidades:** Criação de dashboards, relatórios, análises ad-hoc, entendimento das métricas de negócio e comunicação dos resultados para os stakeholders.
*   **Habilidades Comuns:** SQL, Excel, Ferramentas de BI (Tableau, Power BI, Looker), conhecimentos estatísticos básicos, forte comunicação e entendimento de negócio.
*   **Foco:** O quê aconteceu? Por que aconteceu? (Análise Descritiva e Diagnóstica).

## Engenheiro de Dados (Data Engineer)

O Engenheiro de Dados é responsável por construir e manter a infraestrutura e a arquitetura que permitem a coleta, armazenamento e processamento de grandes volumes de dados.

*   **Responsabilidades:** Criação de pipelines de dados (ETL/ELT), modelagem de dados, manutenção de data warehouses e data lakes, garantia da qualidade, segurança e disponibilidade dos dados.
*   **Habilidades Comuns:** SQL avançado, Python, Scala ou Java, ferramentas de Big Data (Spark), orquestradores (como Airflow), cloud computing (AWS, GCP, Azure) e ferramentas de transformação (como dbt).
*   **Foco:** Como podemos disponibilizar os dados de forma confiável e escalável?

## Cientista de Dados (Data Scientist)

O Cientista de Dados utiliza técnicas avançadas de estatística, matemática e ciência da computação para prever tendências futuras e resolver problemas complexos através de modelos matemáticos e machine learning.

*   **Responsabilidades:** Desenvolvimento de modelos preditivos, algoritmos de machine learning, experimentação (Testes A/B), análises preditivas e prescritivas.
*   **Habilidades Comuns:** Python ou R, bibliotecas de machine learning (Scikit-learn, TensorFlow, PyTorch), estatística avançada, cálculo, álgebra linear e SQL.
*   **Foco:** O que vai acontecer? Como podemos otimizar o resultado? (Análise Preditiva e Prescritiva).

---

### Resumo prático

*   **Engenheiro de Dados:** Constrói as "tubulações" e armazena os dados.
*   **Analista de Dados:** Olha para os dados organizados para explicar o passado e o presente ao negócio.
*   **Cientista de Dados:** Usa os dados e matemática avançada para prever ou otimizar o futuro.
